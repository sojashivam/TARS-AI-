Tars AI Documentation

Introduction

Welcome to the Tars AI documentation. Here you'll find comprehensive information on how to effectively use our AI-powered document analysis service, understand its features, and troubleshoot common issues.

Tars AI allows you to upload PDF documents and images, then interact with their content through a conversational chat interface to extract information, get summaries, and gain insights quickly.

Getting Started

Using Tars AI is straightforward. Follow these simple steps to begin interacting with your documents:

  1. Navigate to the main Tars AI application or chatbot interface page.
  2. Use the designated upload area or button to select your PDF document or image file from your device.
  3. Wait a few moments while Tars AI securely processes and analyzes your document's content. The duration depends on the file size and complexity.
  4. Once processing is complete, the chat interface will become active. Start typing your questions about the document content into the chat input field.

Supported File Types & Limits

Tars AI currently supports the following file types for analysis:

  • PDF documents (.pdf) - Both text-based and scanned (OCR support required, check your plan features).
  • Images (.jpg, .jpeg, .png) - Useful for analyzing diagrams, charts, or scanned pages saved as images.

Please note the following limitations (these may vary based on your subscription plan):

  • Maximum File Size: Typically around 10MB - 50MB. Refer to your plan details.
  • Maximum Page Count: Limits may apply per document (e.g., 20 pages for Free, 100 pages for Pro).

Asking Effective Questions

To get the most accurate and relevant answers from Tars AI, consider these tips when formulating your questions:

  • Be Specific: Instead of "What is this document about?", ask "What are the key findings mentioned in the conclusion section?".
  • Reference Context: If possible, mention page numbers or section titles, e.g., "Summarize the main points on page 5." or "What methodology is described in Chapter 3?".
  • One Question at a Time: Avoid asking multiple unrelated questions in a single message for clearer responses.
  • Start Broad, Then Narrow: Ask a general question first (e.g., "What is the main topic?") to understand the structure, then follow up with specific queries.
  • Use Keywords: Include relevant keywords from the document in your question.

Troubleshooting Common Issues

If you encounter problems while using Tars AI, here are some steps to try:

  • Document Legibility: Ensure the source document (especially scanned PDFs or images) is clear and legible. Poor quality scans can hinder accuracy.
  • File Size/Page Limits: Verify that your file does not exceed the maximum size or page count allowed by your current plan.
  • File Corruption: The uploaded file might be corrupted. Try opening it locally or uploading a different version or file.
  • Processing Errors: Occasionally, processing might fail. Try re-uploading the document after a short wait.
  • Unclear Answers: If the AI's answer seems incorrect or vague, try rephrasing your question with more specific details or context.
  • Contact Support: If problems persist after trying these steps, please reach out to our support team via the Contact page, providing details about the issue and the document (if possible).

API Access

For developers looking to integrate Tars AI's capabilities into their own applications or workflows, we offer API access on specific plans (typically Enterprise).

Please refer to the dedicated API Documentation page (if available) or contact our sales team through the Contact page for details on API endpoints, authentication, usage limits, and pricing.

Security & Privacy

We prioritize the security and privacy of your documents and data.

  • Documents are processed over secure connections (HTTPS).
  • Uploaded documents are typically stored temporarily for processing and analysis during your active session and may be deleted afterwards according to our data retention policy.
  • We do not use your document content to train our general AI models unless explicitly stated and agreed upon.
  • Refer to our official Privacy Policy and Terms of Service for detailed information on data handling and security measures.
